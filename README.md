# 投资分析工具

## 项目简介

投资分析工具是一个全栈Web应用，用于分析股票、基金和金属价格数据，提供详细的财务信息和投资建议。

## 技术栈

### 前端
- React 18
- Chart.js / react-chartjs-2
- CSS3 (Flexbox, Grid)
- Vite

### 后端
- Node.js
- Express.js
- CORS middleware

## 核心功能

### 1. 股票分析
- 股票基本信息（所属行业、主营业务、主要产品）
- 估值指标（PE、PB、ROE、股息率、盈利收益率）
- 财务报表（资产负债表、利润表、现金流表）
- 公司简介
- 历史股价走势（支持年份切换）
- 投资建议

### 2. 基金分析
- 基金基本信息
- 历史净值走势（支持年份切换）
- 投资建议

### 3. 金属价格分析
- 黄金、白银、铜的实时价格
- 历史价格走势（支持年份和月份切换）
- 金属简介

## API设计

### 1. 股票API
- `GET /api/stock/:code` - 获取股票数据

### 2. 基金API
- `GET /api/fund/:code` - 获取基金数据

### 3. 金属API
- `GET /api/metal/:code` - 获取金属价格数据

### 4. 投资建议API
- `POST /api/advice` - 生成投资建议

## 本地开发

```bash
cd backend
npm install
node index.js
```

后端服务将运行在 http://localhost:3001

### 前端启动

```bash
cd frontend
npm install
npm run dev
```

前端服务将运行在 http://localhost:5173

## 功能使用说明

### 1. 股票分析
1. 在搜索页选择"股票"标签
2. 输入股票代码或点击示例代码
3. 点击"搜索"按钮
4. 在详情页查看股票信息、财务报表和历史股价走势
5. 点击年份按钮切换历史数据

### 2. 基金分析
1. 在搜索页选择"基金"标签
2. 输入基金代码或点击示例代码
3. 点击"搜索"按钮
4. 在详情页查看基金信息和历史净值走势
5. 点击年份按钮切换历史数据

### 3. 金属价格分析
1. 在搜索页点击"黄金"、"白银"或"铜"标签
2. 系统将自动跳转到对应金属的详情页
3. 在详情页查看金属价格和历史走势
4. 点击年份和月份按钮切换历史数据

## 投资建议指标说明

### 市盈率（PE）
- PE < 15：低估，投资价值高
- 15 ≤ PE ≤ 25：合理，投资价值适中
- PE > 25：高估，投资价值低

### 市净率（PB）
- PB < 1.5：低估，投资价值高
- 1.5 ≤ PB ≤ 3：合理，投资价值适中
- PB > 3：高估，投资价值低

### 股息率
- 股息率 > 3%：良好，分红回报高
- 股息率 ≤ 3%：一般，分红回报适中

### 盈利收益率（1/PE）
- 盈利收益率 > 6%：良好，投资回报潜力大
- 盈利收益率 ≤ 6%：一般，投资回报潜力适中
